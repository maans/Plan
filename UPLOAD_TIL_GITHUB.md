# Upload til GitHub

1. Pak ZIP-filen ud.
2. Upload hele mappen `plan-library` til roden af repositoryet `Plan`, ved siden af `index.html`.
3. Bevar mappestrukturen og filnavnene.
4. Vent på, at GitHub Pages har udgivet ændringerne.
5. Åbn Plan og vælg **Hent demoforløb**.

Direkte kontroladresser efter udgivelse:

- `plan-library/catalog.json`
- `plan-library/dansk/dansk-2025-26/forloeb.json`
- `plan-library/andre/faellessamlinger-2026-27/forloeb.json`
